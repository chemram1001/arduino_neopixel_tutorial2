Arduino: NeoPixel LEDs Tutorial

By Ramces Gonzalez

May 29, 2019

The goal of this tutorial is to introduce you to the neoPixel stick for Arduino. The neoPixel stick usually comes with 8 LEDs connected in a line. The materials you will need for this tutorial are shown in the list below:

  • Arduino Uno
  • jumper wires
  • NeoPixel stick (8 LEDs)

Once you have the materials, you need to download the library called Adafruit_NeoPixel which is found in GitHub (Adafruit_NeoPixel Library).

Since the neopixel stick does not have any wires, you will have to solder some wires to the connections shown in the image above. You can take some male-male wires and solder one end of one to GND, VCC, and VIN, in the neoPixel stick. (Note: the other GND does not need to be soldered)

The setup is very easy for this tutorial: simply make the following connections with jumper wires:

  • Arduino ==========> NeoPixel Stick
  • Pin 13 ==========> VIN
  • 5V ==========> VCC
  • GND ==========> GND

Also, in the loop() function, note that there are 2 for loops, the first one loops through the first to last LED (pixel), since there are 8 pixels in a single neoPixel stick, the for loop runs from the first pixel which is pixel 0, all the way to the last pixel which is pixel 7, that is why the terminating statment is at 'i < NUMPIXELS'. This first for loop will turn each pixel in the array into a medium green color which is given by 'rgb(0,150,0)'. Also, within this loop there is an if-else statement which makes an alterating green-blue color pattern by assinging odd pixels blue and even pixels green. Next, the second for loop turns off the pixels from the end of the array to the beginning by setting pixels.Color(0,0,0).

Once you upload and verify that the setup and the code works, make sure to modify play around with the code in order to become more familiar with the neoPixel stick. Also, you can read more about the Adafruit_NeoPixel library from the GitHub page.
